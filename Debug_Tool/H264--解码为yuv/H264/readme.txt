    很多人经常会在我们的论坛（ffmpeg工程组：http://bbs.chinavideo.org）中

提问，寻找哪儿有H.264解码器，询问如何从ffmpeg中提取H.264的解码器，询问

是否有VC下的工程，可以实现H.264的解码，……，为了满足这些朋友的要求，

ffmpeg工程组成员“hust_xcl”特地从ffmpeg中提取出了H.264的解码器部分（H.264

的编码器有独立的工程：x264），并用VC++生成了工程文件，各位有需要的朋友

可以在本工程的基础上进行适当的修改，以满足自己的需求。



    该工程的编译环境：
    1、Visual Studio 6.0 + SP5（http://msdn2.microsoft.com/en-us/vstudio/aa718363.aspx）
    2、Visual C++ 6.0 Processor Pack（http://download.microsoft.com/download/vb60ent/update/6/w9x2kxp/en-us/vcpp5.exe）
    （注意：Visual C++ 6.0 Processor Pack在VC6.0+SP6的环境下会安装失败）

    在编译、应用该工程的过程中，出现的相关问题，请在我们的论坛寻找解决方案，

或是与“hust_xcl”直接联系：hust_xcl@126.com。
